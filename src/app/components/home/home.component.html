Aplicación para gestionar proyectos

Se propone desarrollar un Sistema de Información web para el registro, seguimiento y control de los proyectos académicos (docencia, investigación y extensión) para el departamento de Ingeniería de sistemas. Los proyectos de docencia pueden incluir talleres, laboratorios, prácticas, proyectos integradores, etc.

Este es un desarrollo para la asignatura Proyecto Integrador I
